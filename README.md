# Introduction

This workshop will explain how to use the benefits of [Infrastructure-as-Code](https://en.wikipedia.org/wiki/Infrastructure_as_Code) with tools like [Ansible](https://www.ansible.com/) & [Vagrant](https://www.vagrantup.com/) to install and configure a fully featured GitLab instance. This will include [GitLab CI](https://about.gitlab.com/features/gitlab-ci-cd/), [Docker](https://www.docker.com/), domain configuration incl. generation of valid [LetÂ´s Encrypt](https://letsencrypt.org/) certificates, [GitLab Container Registry](https://docs.gitlab.com/ee/user/project/container_registry.html) & [GitLab Runner](https://docs.gitlab.com/runner/) configuration.

{% hint style="info" %}
 This workshop assumes that you have a workstation with 12+ GB of RAM and local admin rights.
